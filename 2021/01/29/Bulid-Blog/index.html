# First Blog

# 关于使用 hexo 和 github 的 pages 来进行博客制作的个人操作部分，会逐渐补全整个过程！

# 需要安装

  1. node.js 环境
  1. (https://nodejs.org/zh-cn/) 下载稳定版 (长期支持版) 即可
  2. 安装一路 Next 即可，在选择安装目录时，可以自由选择
  3. 安装完成，打开 cmd，输入 node -v 命令，如果显示版本号即证明安装成功
  1. github 注册
  1. (https://github.com/) sing up 注册即可
  2. 点击右上角头像下有 Create a new repository 点击并进行创建仓库
  3. 仓库名一定要以 注册时的用户名.github.io 形式 (例如：qiliang1015.github.io)，然后创建即可

此处可以进行一个测试

  1. 点击 Create new file 创建一个文件，命名为 index.html，并且写入
  1. 然后在浏览器地址栏输入前面的仓库名，出现这是我的主页字样即可
  1. git 安装
  1. (https://git-scm.com/download/win) 安装 git 工具
  2. 安装一路 Next, 有 Use Windows'default console window 选项记得勾选，在选择安装目录时，可以自由选择
  3. 安装完成，打开 cmd，输入 git 命令，如果显示相关命令帮助，即证明安装成功
  1. github ssh 配置
  1. 进入 git 安装目录下，打开 git-bash.exe 程序
  2. 输入命令 ssh-keygen -t rsa -C "邮箱" 回车即可，不需输入其他东西
  3. 打开 github，点击头像，进入 settings (设置), 点击 SSH and GPG keys
  4. 点击 New SSH，title 任意填写
  5. 进入电脑的 系统盘 / 用户 / 当前电脑用户名 文件夹下，寻找.ssh 文件打开，进入并以记事本形式打开名为 id_rsa.pub 的文件，全部复制
  6. 然后到 github 中新建的 ssh 中的 key 中粘贴即可，然后 Add SSH key
  1. hexo 的本地使用
  1. (https://hexo.io/zh-cn/docs/) 搜索进入此网址
  2. 本地新建一个存放博客的文件夹，打开 cmd 命令行，cd 到博客文件夹下执行 hexo 文档中的安装命令 npm install hexo-cli -g
  3. 输入 hexo init 博客名 进行初始化
  4. cd 到博客目录
  5. 安装依赖，即输入 npm install
  6. 输入 hexo s 即可在本地运行，可以输入 -p 5555 等后缀改变访问端口，以防因端口占用问题出现错误等
  7. 浏览器输入 localhost:5555 访问，即可看见 hexo 的默认博客

  1. 安装任意编辑器一个，以供修改文件等，推荐 vscode，可直接搜索下载安装

  2. hexo 发布到 github

  1. 以编辑器打开博客目录下的 _config.yml 文件
  2. 打开 github，进入仓库，将仓库下载的链接进行复制
  3. 在 yml 文件最下方，有 deploy 属性，在此属性下再添加一条 reop:仓库地址 和 branch:master
  4. 在上方寻找 url 属性，将其 // 后的内容修改成仓库名称，(如：https://qiliang0102.github.io)
  5. 在博客目录下通过 cmd 输入 npm install hexo-deployer-git --save 安装此插件
  6. 然后输入 hexo g 生成
  7. 然后输入 hexo d 提交到 GitHub
  8. 会提示配置 github 信息
    输入 git config --global user.email "邮箱" 然后回车 输入 git config --global user.name “用户名” 然后回车
  9. 输入 hexo d 提示输入 github 用户名密码，输入即可
  10. 命令行最后显示 INFO Deploy done:git 即成功
  11. 浏览器访问之前的仓库名 (如：qiliang0102.github.io)，即可看到 hexo 博客主页

# hexo 主题预览

(https://hexo.io/themes) 主题地址

  1. 选择自己喜欢的主题，打开可以预览

# hexo 之 next 主题使用

  1. 主题官网搜索 next，进入 github 源码，将主题下载下来
  2. 进入博客目录的.config.yml 文件 寻找 theme 属性，将值修改成主题名，如： theme:next
  3. 将下载的主题文件 (若是压缩包，请自行解压)，文件夹名称修改成 next，将文件夹复制到博客目录下的 themes 文件夹中
  4. 命令行 cd 到博客目录下，输入 hexo g 生成，然后输入 hexo s 本地预览下，可以看到主题已经有变化了
  5. 具体有关于 next 模板的使用及内容修改请查看 next 主题文档 (https://theme-next.iissnan.com/)
  6. 经由修改等操作后，即可 hexo g -d 上传到 github

以上即是 hexo+github 创建博客的流程，需要一些基础知识，所有有关文件设置的内容，请千万记得保存！
关于以上流程有任何疑问，可留言到文章下方！
