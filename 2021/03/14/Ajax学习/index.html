# Ajax

# Ajax 简介

Ajax 就是异步的 JS 和 XML。

# Ajax 特点

# Ajax 优点

  1. 可以不用刷新就提交请求并获得结果。
  2. 可以根据事件更新部分页面内容。

# Ajax 缺点

  1. 没有浏览记录，不可以回退。
  2. 存在跨域问题（也就是不能往另一个网页去提交请求）。
  3. seo（搜索引擎优化）不友好，没办法通过爬虫 / 源代码之类的获取到内容，是通过 ajax 请求并获取结果通过 js 动态创建的。
