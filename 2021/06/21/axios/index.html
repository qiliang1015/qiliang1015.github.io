# Axios

axios 基于 promise 对 Ajax 的封装

# 入门

  1. 使用方式
  1. npm 安装
    npm install --save axios
  2. cdn 引入
    <script src="https://unpkg.com/axios/dist/axios.min.js"></script>
  1. 简单的 axios 请求

默认方式请求 (无参)

默认方式 GET 请求 (无参)

默认方式 GET 请求 (有参)
将参数附带在 url 地址后，需要添加？，然后将参数以键值对附带

默认方式 POST 请求 (无参)

默认方式 POST 请求 (有参)
默认的 axios 有参 POST 请求 data 是发送 application/json 格式
可以修改成 params 参数发送方式，实际是以连接到 url 后方式发送

  1. 使用 get ()，post () 方式请求

使用 axios.get () 发送无参请求

使用 axios.get () 发送有参请求

使用 axios.post () 发送有参请求

两类拦截器
1. 请求方向 (成功 / 失败)

# 拦截器的作用

> 在网络请求在请求之前或者响应时，对操作进行相应处理
  1. 发起请求时，添加网页加载动画
  2. token 认证时，可以请求之前，可以做判断看是否有 token，然后进行相应操作，例如要求登录等
  3. 响应时，可以进行相应的数据处理
