# ES6 学习

为什么学习 ES6

  1. 更高的开发效率 Object.assign Array.from …
  2. 更少的出错可能 let&const Symbol …
  3. 更人性化的语法 箭头函数 async&await …

# 使用脚手架

  1. npm install learn-es-cli -g 全局安装脚手架
  2. learn-es-cli init 初始化项目 (es6-project)
  3. cd es6-project 进入项目列表
  4. npm install 安装所需依赖
  5. npm run dev 运行项目
  6. http://localhost:8080 通过给定端口访问 (端口不尽相同，依据给定访问)

# babel 简单使用

转换 es6 -> es5
npm install babel-cli -g 安装 babel
npm install -D babel-preset-es2015 安装转 es6 依赖
创建 src 文件夹，并写段 es6 语法 js 代码
创建.babelrc 文件，写入规则 {"presets": ["es2015"]} 配置转换规则
bebel src -d dist 转换命令

  • 不允许重复声明

  • 不存在变量提升

  • 暂时性死区

  • 块级作用域

# Const

同 let 一样

  • 不属于顶层对象 window
  • 不允许重复声明
  • 不存在变量提升
  • 块级作用域

需要注意的是:
const 声明的变量必须进行初始化，不然会抛出异常 Uncaught SyntaxError: Missing initializer in const declaration。

const 实际上保证的并不是变量的值不能改动，而是变量指向的那个内存地址所保存的数据不能改动。
