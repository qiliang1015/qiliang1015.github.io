# 测试题小集合

  1. 数组有哪些原生方法

    • Array.join () 方法是将一个数组里面的所有元素转换成字符串，然后再将他们连接起来返回一个新数组
    • Array.reverse () 方法将颠倒数组中元素的顺序并返回一个颠倒后的数组
    • Array.sort () 是在原数组上进行排序，返回排序后的数组 按照字符顺序
    • rray.slice (a,b) 方法可从已有的数组中返回指定的一个片段 (slice) 两个参数 a 决定从何开始取，b 决定从何结束取
    • Array.concat () 方法用于连接两个或多个参数（数组，字符串等
    • Array.splice (index,howmany,item) 方法从数组中添加 / 删除元素，然后返回被删除的元素。它在原数组上修改数组 三个参数 index 规定操作的位置 howmany 要操作的元素数量 (默认从 index 开始到结束) item 要添加的新元素
    • Array.push () 方法可向数组的末尾添加一个或多个元素，并返回新的长度
    • Array.pop () 方法用于删除并返回数组的最后一个元素
    • Array.forEach () 为每个数组元素执行 callback 函数
    • Array.map 方法会给原数组中的每个元素都按顺序调用一次 callback 函数，callback 每次执行后的返回值（没有指定返回值则返回 undefined）组合起来形成一个新数组。
    • Array.filter () 为数组中的每个元素调用一次 callback 函数，并利用所有使得 callback 返回 true 或 等价于 true 的值 的元素创建一个新数组

  2. JavaScript 中的’use strict’

    严格模式，使用后代码会在语法等方面进行严格检测

    1. 严格模式通过抛出错误来消除了一些原有静默错误。
    2. 严格模式修复了一些导致 JavaScript 引擎难以执行优化的缺陷：有时候，相同的代码，严格模式可以比非严格模式下运行得更快。
    3. 严格模式禁用了在 ECMAScript 的未来版本中可能会定义的一些语法。

    • 传值，将值 copy 出来赋值，赋值完成后，被赋值变量发生改变，原变量不受影响
    • 传址，将变量指向另一个变量的地址，两者共同享有一个地址区域，无论任意一个进行地址内数据的修改，都受影响
